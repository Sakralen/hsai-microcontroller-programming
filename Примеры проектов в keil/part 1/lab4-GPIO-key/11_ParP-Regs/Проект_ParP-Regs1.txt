    Проект ParP-Regs1

    Используется в 1-ом занятии Intro-части курса.

    Назначение: 
  Демонстрация простейшего проекта.
В проекте пока три конфигурации, в них используется прямое обращение к регистрам ППс. 
ParP-Regs-D для Discovery, - мигает диодом, по нажатию кнопки USER останавливается
ParP-Regs-N для Nucleo, - мигает диодом, по нажатию кнопки USER останавливается
ParP-Regs-D для Discovery, - подключен динамик, мигает диодом, 
                  по нажатию кнопки перекл в генерацию звука. 

    Что делает программа:
Периодически переключает выход порта со СвДиодом через время, определяемое
программной задержкой void Delay(u32 uiTDel). 
Опрашивает состояние входа, к которому подключена кнопка USER. 
Если кнопка нажата, переключение СвДиода приостанавливается.

8) Что такое "стандартные библиотеки"
